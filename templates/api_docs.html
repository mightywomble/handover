Onboard Customer

This endpoint allows you to programmatically onboard a new customer.

POST /api/onboard/customer

Example `curl` Request

curl -X POST -H "Content-Type: application/json" -d '{
    "account_manager": "John Doe",
    "salesforce_reference": "SF-12345",
    "company_name": "API Test Corp",
    "address_1": "123 API Lane",
    "city": "Techville",
    "contacts": [
        {"First Name": "Jane", "Last Name": "API", "Email": "jane@apitest.com"}
    ]
}' {{ request.url_root }}api/onboard/customer

Example JSON Payload

{
    "account_manager": "John Doe",
    "salesforce_reference": "SF-12345",
    "company_name": "API Test Corp",
    "address_1": "123 API Lane",
    "address_2": "",
    "city": "Techville",
    "state": "CA",
    "zip_code": "90210",
    "country": "USA",
    "phone": "555-0101",
    "notes": "Customer onboarded via API.",
    "contacts": [
        {
            "First Name": "Jane",
            "Last Name": "API",
            "Email": "jane@apitest.com",
            "Phone": "555-0102",
            "Company": "API Test Corp",
            "Dept": "IT",
            "Is Supervisor": "No"
        }
    ]
}

Onboard Supplier

This endpoint allows you to programmatically onboard a new supplier.

POST /api/onboard/supplier

Example `curl` Request

curl -X POST -H "Content-Type: application/json" -d '{
    "supplier_name": "API Supplier Inc.",
    "ci_type": "Off Platform",
    "status": "Production Available"
}' {{ request.url_root }}api/onboard/supplier

Example JSON Payload

{
    "supplier_name": "API Supplier Inc.",
    "ci_type": "Off Platform",
    "ci_sub_type": {
        "selection": ["Other"],
        "ci_sub_type_other": "Custom Hardware"
    },
    "status": "Production Available",
    "supplier_dc": "us-east-99",
    "dc_address": "99 API Road",
    "primary_contact_name": "Supplier Support",
    "primary_contact_email": "support@apisupplier.com",
    "support_contact_method": {
        "selection": ["Email", "Portal"]
    },
    "support_email": "support@apisupplier.com",
    "support_hours": {
        "selection": ["Other"],
        "support_hours_other": "9am-5pm PST M-F"
    },
    "on_site_tech_name": "N/A"
}
